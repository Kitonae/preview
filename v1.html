Coordinate System

Precision Coordinate Mapping

WATCHOUT 6 and 7 use different coordinate conventions. The importer handles every transformation automatically.

  • Y/Z axis negation for 3D space compatibility
  • Anchor point flipping relative to image height
  • Scale and opacity normalization (0–1 → 0–100)
  • Composition reference frame adjustment
  • Projector eye, origin, and frustum conversion
  • Layer index and MIDI channel re-indexing
  • Bezier control point coordinate transforms

Technology

Built for Reliability

Written in Rust for memory safety and performance, with a Vue 3 UI for real-time progress tracking.

🦀 Rust Core

The wo6-importer crate provides type-safe parsing, OID reference resolution, and comprehensive error handling across two JSON dialects.

🔌 Tauri Integration

Seamless IPC between the Rust backend and Vue 3 frontend with real-time Wo6ImportProgress and Wo6ImportComplete events.

🚀 Asset Pipeline

Direct integration with the WATCHOUT 7 asset server via StatelessClient for automatic media registration and folder hierarchy preservation.
